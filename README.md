# tverrec-proot-ubuntu

## 1. Run on Termux

```
pkg update
pkg upgrade -y
pkg install git -y
git clone https://github.com/dongaba/tverrec-proot-ubuntu.git
cd tverrec-proot-ubuntu.git
chmod +x setup-termux.sh
./setup-termux.sh
ubuntu
```

## 2. Run on Ubuntu

```
apt-get update
apt-get upgrade -y
apt-get install git -y
git clone https://github.com/dongaba/tverrec-proot-ubuntu.git
cd tverrec-proot-ubuntu.git
chmod +x setup-ubuntu.sh
./setup-ubuntu.sh
```
